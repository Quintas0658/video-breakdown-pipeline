## 目标 / Goal

用 4 周时间发布 8-10 条视频，验证"视频+笔记双语学习"的市场需求和内容形式。

---

## 第 0 周：基础搭建（Day 0-2）

### 0.1 频道设立

| 平台 | 行动 | 备注 |
|:---|:---|:---|
| **YouTube** | 创建频道，设定品牌名、头像、Banner、About | 主战场，长视频 5-10 分钟 |
| **TikTok** | 创建账号，同品牌名 | 短视频 30-60 秒引流 |
| **小红书** | 创建账号（如果要测中国市场） | 可选，同内容中文版 |

> [!TIP]
> **brand-storytelling**: *"Story before product"* — 先想清楚频道讲什么故事，再做视觉。你的故事是：**"我帮你打破语言壁垒，用 10 分钟搞懂一个专业视频"**

### 0.2 品牌名 & 定位语

需要确认：
- 频道名（英文为主，要有辨识度）
- 一句话定位：`"Professional growth without language barriers"` 或你想调整的版本
- 视觉风格：配色、字体、封面模板

### 0.3 内容模板制作

**每条视频的结构** *(brand-storytelling: "Hook, message, celebration")*：

```
[0:00-0:15]  🎣 HOOK — 挣扎时刻
             "Have you ever watched a sales video and understood 
              the words but NOT the strategy behind them?"

[0:15-1:00]  📋 SETUP — 今天拆解什么
             "Today I'm breaking down [Expert]'s cold call technique
              so you learn BOTH the language AND the skill"

[1:00-7:00]  🎬 BREAKDOWN — 核心内容（分 2-3 个片段）
             每个片段：播放原视频 → 暂停 → 白板笔记讲解
             • Key phrase + 翻译/解释
             • 为什么这样说（文化/策略上下文）
             • 替代表达 / 常见错误

[7:00-8:00]  📝 RECAP — 总结所有句型
             一张"速查表"画面

[8:00-8:30]  🎯 CTA — 行动号召
             "Save this for your next call. Comment which topic 
              you want next. Subscribe for weekly breakdowns."
```

> [!IMPORTANT]
> **user-onboarding**: *"Make the first 30 seconds magical"* — 前 15 秒决定观众走不走。用**具体的挣扎场景**开头，不要用 "Hi everyone, welcome to my channel"

## 第 1 周：第一条视频（Day 3-9）

### 1.1 选片 & 拆解

**步骤：**
1. 从 Cold Call Masterclass 选一个 2-3 分钟的精华片段
2. 逐句拆解：
   - 🔤 关键英文表达 + 中文/西语翻译
   - 💡 为什么这样说（策略/文化原因）
   - ❌ 常见错误表达
   - ✅ 可直接复制的替代说法
3. 写成脚本（含画面指引）

### 1.2 录制 & 剪辑

**工具推荐：**

| 工具 | 用途 | 价格 |
|:---|:---|:---|
| **CapCut / 剪映** | 剪辑、字幕、画中画 | 免费 |
| **Canva** | 封面图、白板笔记画面 | 免费/Pro |
| **OBS Studio** | 录屏 + 画中画（如果做直播风格） | 免费 |
| **Descript** | AI 字幕 + 快速剪辑 | 免费额度 |

**画面构成：**
```

│   │  🎬 原视频   │ │ 📝 你的   │  │

│                                   │
└───────────────────────────────────┘
```

### 1.3 发布第一条视频

**YouTube 发布清单：**
- [ ] 标题：用"数字+对比+好奇心" 公式 — *"5 Cold Call Phrases That Sound Native (Not What Duolingo Teaches)"*
- [ ] 描述：前 2 行包含关键信息 + 时间戳 + 相关链接
- [ ] Tags：`business english`, `cold calling`, `professional english`, `language learning`, `sales english`
- [ ] 封面图：对比风格（Before/After 或 ❌ vs ✅）
- [ ] End screen + Cards 指向下一个视频

**TikTok 版本：**
- [ ] 从 YouTube 视频里截取最精华的 30-60 秒
- [ ] 竖屏重新排版
- [ ] 加字幕（TikTok 观众很多静音观看）
- [ ] Hashtags：`#businessenglish #coldcalling #languagelearning #professionaldevelopment`

> [!TIP]
> **launch-marketing**: *"Optimize for algorithm momentum"* — 发布后 24 小时内把链接分享给朋友/群组，让算法看到初始互动信号。不需要很多人，10-20 个真实观看+互动就够了。

## 第 2-3 周：建立节奏（2 条/周）

### 2.1 内容日历

| 周 | 视频 1（周二发） | 视频 2（周五发） |
|:---|:---|:---|
| **Week 2** | "How to Handle 'I'm Not Interested' Like a Pro" | "Professional English vs Textbook English: Cold Call Edition" |
| **Week 3** | "The Opening Line That Gets a 40% Response Rate" | "I Broke Down a Sales Master's Script — Here's What I Learned" |

> **content-marketing**: *"Consistency beats production value"* — 不要追求完美，每周稳定出 2 条比 1 条完美视频更重要。

### 2.2 增长飞轮启动 *(designing-growth-loops)*

```
发布视频 → 观众保存/分享 → 算法推送 → 新观众 → 更多互动
    ↑                                              │
    └──────────── 你看评论，发现新需求 ────────────────┘
```

**每条视频发布后做的事：**
1. 分享到 3-5 个相关社群（Reddit r/languagelearning, r/sales, LinkedIn groups）
2. 回复每一条评论（前 100 条的回复率要 100%）
3. 记录评论中的需求信号（"能做 XX 主题吗？"）

> **community-building**: *"Go where the community already is"* + *"Start small with the most vocal"* — 找到 Reddit/LinkedIn/Discord 上已经在讨论这个话题的人，而不是建一个空群等人来。

### 2.3 品牌叙事进化 *(brand-storytelling)*

> *"You're Obi-Wan, not Luke"* — 你是导师，用户是英雄。

在视频中强化：
- ❌ "I'm going to teach you..." → ✅ "Let's break this down together so YOU can use it tomorrow"
- ❌ 展示你多厉害 → ✅ 展示用户用了之后多厉害
- 分享一个自己学语言的"挣扎故事"建立共鸣

## 第 4 周：看数据 + 测试付费

### 4.1 关键指标回顾 *(measuring-product-market-fit)*

| 指标 | 好信号 | 坏信号 | 行动 |
|:---|:---|:---|:---|
| **平均观看时长** | > 50% | < 30% | 调整 hook 或缩短视频 |
| **保存率** | > 5% | < 2% | 内容不够实用，换角度 |
| **评论质量** | "能做 XX 吗？" 具体需求 | 只有 "nice video" | 还没戳到痛点 |
| **回头观众** | > 30% 看了 2 条以上 | < 10% | 内容没有连贯价值 |
| **订阅转化率** | > 3% 观看→订阅 | < 1% | 定位或 CTA 需调整 |

### 4.2 首次付费测试 *(pricing-strategy + founder-sales)*

> *"Sell before you build"* + *"WTP is a proxy for value"*

**测试方式：**

```
视频描述/Pinned comment 中放链接：

"📥 Download the complete Cold Call Phrase Pack (PDF)
 — All 25 phrases from this series with notes, 
   translations, and practice exercises
 → [Gumroad link] — $5"
```

**如果有人买** → PMF 信号，继续打磨
**如果没人买** → 没关系，看下载免费版的人数，调整定价或内容

### 4.3 习惯养成设计 *(behavioral-product-design)*

> *"Once you hit seven days, loss aversion kicks in"*

在视频中植入习惯循环：
- **Cue（触发）**：每周固定时间发布 → 观众形成期待
- **Routine（行动）**：打开视频，跟着做笔记
- **Reward（奖励）**：视频最后的 Recap 给"我学到了"的满足感
- **Investment（投入）**：评论区留下自己用这些句型的经历 → 社交承诺
